Getting Started

Welcome to the Aoede API! Our REST API allows you to integrate powerful AI-driven code generation capabilities into your applications. This documentation will guide you through everything you need to know to get started.

Base URL

https://api.aoede.dev/v1

Quick Start

Here's a simple example to get you started with generating code using our API: 

curl -X POST "https://api.aoede.dev/v1/generate" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Create a Python function to calculate fibonacci numbers",
    "language": "python",
    "framework": "none"
  }'

Authentication

Aoede API uses API keys for authentication. You can obtain your API key from your dashboard after creating an account.

Important: Keep your API keys secure and never expose them in client-side code.

API Key Authentication

Include your API key in the Authorization header with the Bearer scheme: 

Authorization: Bearer YOUR_API_KEY

Example Request

import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.aoede.dev/v1/user', headers=headers)
print(response.json())

Code Generation

The code generation endpoint is the core of Aoede's functionality. It allows you to generate high-quality code in multiple programming languages using advanced AI models.

Request Parameters

Parameter Type Required Description
prompt string Yes Description of the code to generate
language string Yes Target programming language
framework string No Specific framework or library to use
complexity string No Code complexity level (simple, medium, complex)
model string No Specific AI model to use

Example Request

const response = await fetch('https://api.aoede.dev/v1/generate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: 'Create a REST API endpoint for user authentication',
    language: 'python',
    framework: 'fastapi',
    complexity: 'medium'
  })
});

const result = await response.json();
console.log(result);

Project Management

Organize your code generations into projects for better management and collaboration.

Create a Project

curl -X POST "https://api.aoede.dev/v1/projects" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Web App",
    "description": "A modern web application",
    "language": "typescript",
    "framework": "react"
  }'

List Projects

import requests

headers = {'Authorization': 'Bearer YOUR_API_KEY'}
response = requests.get('https://api.aoede.dev/v1/projects', headers=headers)

for project in response.json()['data']:
    print(f"Project: {project['name']} - {project['status']}")

Error Handling

Aoede uses conventional HTTP response codes to indicate the success or failure of an API request. In general: Codes in the 2xx range indicate success. Codes in the 4xx range indicate an error that failed given the information provided. Codes in the 5xx range indicate an error with Aoede's servers.

200

OK

Everything worked as expected.

400

Bad Request

The request was unacceptable.

401

Unauthorized

No valid API key provided.

429

Too Many Requests

Too many requests hit the API too quickly.

Error Response Format

{
  "error": {
    "type": "invalid_request_error",
    "code": "missing_parameter",
    "message": "The 'prompt' parameter is required for code generation.",
    "param": "prompt"
  }
}

Rate Limits

The Aoede API implements rate limiting to ensure fair usage and maintain service quality for all users.

Rate Limit: 100 requests per minute for free tier, 1000 requests per minute for pro tier.

Rate Limit Headers

Every API response includes headers that provide information about your current rate limit status: 

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 85
X-RateLimit-Reset: 1640995200
